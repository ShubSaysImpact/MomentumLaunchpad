# **App Name**: Momentum Launchpad

## Core Features:

- Sidebar Navigation: Collapsible sidebar for navigating between Dashboard, Zone of Impact, Traction & Buy-in, Monetization, Retrospective, Learnings, and Pin Board pages.
- Goal/Task Management: System to add, track, and complete goals and tasks, categorized as 'daily', 'weekly', or 'monthly', specific to a domain (Clarity, Traction, Monetisation) or global.
- Learning Points/Pinned Ideas: Separate system for managing user-generated "Learning Points" and "Pinned Ideas".
- Actionable Steps Generation: AI tool analyzes Gallup Strengths, Mission, Vision, and "Why" inputs on the Zone of Impact page to generate actionable next steps that can be added to user tasks.
- Retrospective Analysis: AI tool analyzes weekly reflections on the Retrospective page to generate key learning points that can be added to the Learnings board.
- Traction Chatbot: Floating AI assistant chatbot that uses conversation history as a tool, to provide advice on building traction.
- Progress Dashboard: Dashboard displaying a radial progress chart summarizing progress across Clarity, Traction, and Monetisation, and aggregated lists of current weekly goals, weekly tasks, and daily tasks.

## Style Guidelines:

- Primary color: Vibrant teal (#008080) to convey innovation and clarity, inspired by the clinician founders.
- Background color: Light gray (#F0F8FF) to create a clean and unobtrusive backdrop.
- Accent color: Warm gold (#D4AF37) to highlight important actions and success, analogous to the teal for a touch of luxury.
- Body and headline font: 'Inter' sans-serif for a modern and neutral look that is suitable for headlines and body text.
- ShadCN UI components and Tailwind CSS will be used for a modern and clean interface.
- Subtle transitions and animations to enhance user experience without being distracting.
